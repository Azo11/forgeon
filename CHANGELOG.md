# Changelog

All notable changes to Forgeon are documented here.

## [1.3.0] - 2026-07-22

### Added

- Added Dry Run mode with a confirmation gate and a preview of files, commands, external actions, and expected results.
- Added rollback references for commit, deployment state, and previous release tracking.
- Added deployment failure handling that preserves evidence and restores the previous version when supported and authorized.

## [1.2.0] - 2026-07-22

### Added

- Added a pre-execution Environment Check for Git, GitHub CLI, authentication, deployment access, required environment variables, and framework detection.
- Added prerequisite status reporting and dependency-blocking behavior.

## [1.1.0] - 2026-07-22

### Added

- Added a mandatory maintenance workflow covering impact analysis, implementation trade-offs, versioning, execution plans, and approval gates.
- Added explicit completion and external-limitation reporting requirements to project rules and documentation.
